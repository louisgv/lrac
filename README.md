# L's React Component Template

[![Storybook](https://cdn.jsdelivr.net/gh/storybookjs/brand@master/badge/badge-storybook.svg)](https://louisgv.ga/lrct)

What's included:

1. typescript (non-strict)
2. story-book
3. changelog generator
4. styled-components
5. styled-system

## Available Scripts

See `package.json`

## Deploy Instructions

```
yarn dist
cd dist
npm publish
```

